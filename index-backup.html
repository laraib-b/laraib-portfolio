01

AI Email Sorter

Problem

Many of my important emails used to get buried in the Gmail inbox. Searching and sorting them manually was a hassle especially if some important email made its way to spam and I never check it.

Built

An n8n automation that reads incoming Gmail messages, classifies each one by category (defined by me) with an OpenAI chat model, and forwards it to the matching Discord channel within a single server using webhooks.

Architecture

Gmail trigger (new email)↓ Text Classifier (OpenAI chat model)↓ Routed into category branches↓ Per-category Discord channels (webhooks)

Skills demonstrated
